Java

Características

  • Orientado a objetos
  • De uso general
  • Multiplataforma
  • Soporte a tarbajo de red
  • Se puede ejecutar en sistemas remotos de manera segura
  • Basado en C++
  • Se puede incorporar a páginas HTML

Entorno de ejecución

Los programas escritos en ava se utilizan para su ejecución la máquina virtual de Java (JVM), pueden correr en forma de aplicaciones de escritorio, en servidores web como servers o e forma de páginas activas (JSP).

C#

Características

  • Orientado a objetos
  • De uso general
  • Multiplataforma
  • Soporte a tarbajo de red
  • Se puede ejecutar en sistemas remotos de manera segura
  • Basado en C++ y Java
  • Se puede incorporar a páginas HTML

Entorno de ejecución

Loa programas escritos en C# comparten el marco de tarbajo de Microsoft conocido como .NET Framework el cual permite que programas en diferentes lenguajes puedan interactuar.
